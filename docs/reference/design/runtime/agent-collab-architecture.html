Agent 协作 — 架构流程图

一个原语（分支间通信）建在事件层上，从后端一路到前端。三层叠在一张图里： ① 通信原语流程 · ② 事件层总线 · ③ 端到端数据流。设计： docs/design/runtime/agent-collaboration.md

怎么读：从上到下三层——① 一次 message_branch 调用的全流程（投递→触发→回送，含 new/fork/已有分支/sources 几种用法 + 防护）； ② 它建在的事件层总线（源/消费者互不认识）；③ 通信结果怎么经 ws.frame 一路到前端的消息流 / DAG / 面板。
颜色：蓝=发起方/前端 · 绿=目标分支/界面去向 · 橙=事件层/总线 · 紫=综合 · 青=系统源/传输 · 红=防护。
