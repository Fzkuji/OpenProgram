# ACT

Execute the planned action on the target UI element.

## What to do

1. Use the action plan from the LEARN step
2. Execute the action using available GUI tools
3. Record exactly what you did and where
4. Note any immediate feedback from the UI (button press, loading, etc.)

## Available action types

- click: click on an element
- double_click: double-click on an element
- type: type text into an input field
- scroll: scroll up or down
- hover: hover over an element
- key: press a keyboard key or combination

## Be precise

Record the exact coordinates or element description you interacted with.
Note whether the UI responded as expected immediately after the action.
