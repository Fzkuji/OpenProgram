# OBSERVE

Take a screenshot of the current screen and analyze what you see.

## What to do

1. Take a screenshot using available tools
2. Identify all visible UI elements: buttons, input fields, menus, text, icons
3. Note the current state of the screen (what app, what screen/page)
4. Determine whether the target element for the task is currently visible

## What to look for

- Interactive elements (buttons, links, dropdowns, checkboxes)
- Text content that helps understand context
- Error messages or notifications
- Loading states or disabled elements
- Modal dialogs or overlays

## Be precise

Describe element positions clearly (top-left, center, bottom-right, etc.).
Note the exact text on buttons and labels.
If the target is not visible, say so clearly.
