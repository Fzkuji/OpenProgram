# VERIFY

Take a screenshot and confirm the action was completed successfully.

## What to do

1. Take a new screenshot
2. Compare with the previous state
3. Confirm whether the intended action produced the expected result
4. Note the current state of the screen after the action

## What counts as success

- The target element changed state (button was clicked, checkbox checked, etc.)
- Navigation occurred (new page or screen loaded)
- A form was submitted and confirmation appeared
- The expected content is now visible

## What counts as failure

- The screen looks identical to before
- An error message appeared
- The wrong element was affected
- The action had no visible effect

## Be honest

If you cannot confirm success, say so clearly.
Do not assume success without visual evidence.
