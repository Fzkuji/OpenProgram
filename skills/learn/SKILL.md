# LEARN

Analyze the current screen state and determine the best action plan.

## What to do

1. Review what was observed on screen
2. Match the task goal to the available UI elements
3. Determine the exact element to interact with
4. Choose the appropriate action type (click, type, scroll, hover, etc.)
5. If you have seen this app before, apply that knowledge

## Decision process

- Is the target element directly visible and accessible?
- Does anything need to happen first (scroll, close a dialog, navigate)?
- Is the element interactive (not disabled, not hidden)?
- What is the most reliable way to target this element?

## Output a clear action plan

Be specific. Instead of "click the button", say "click the blue 'Sign In' button
in the top-right corner of the navigation bar".
